# 🐍 Advanced Snake Game

A modern and advanced version of the classic Snake Game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 🐍 Smooth rounded snake design (no more boxy look)
* 🎮 Keyboard + Mobile controls
* 📈 Level system (speed increases as score grows)
* 🏆 High Score saving using LocalStorage
* 💀 Game Over screen with restart button
* 🔁 One-click Restart Game
* 🎨 Clean and glowing UI design

## 🎮 Controls

### 🖥️ Keyboard:

* ⬆️ Arrow Up → Move Up
* ⬇️ Arrow Down → Move Down
* ⬅️ Arrow Left → Move Left
* ➡️ Arrow Right → Move Right

### 📱 Mobile:

* On-screen buttons for movement

## 🧠 How It Works

* The snake moves continuously on the canvas
* Eat food (red circle) to grow and increase score
* Every 5 points → level increases → speed increases
* Game ends if:

  * Snake hits wall
  * Snake hits itself

## 🛠️ Technologies Used

* HTML5 (Canvas)
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start playing 🎮
